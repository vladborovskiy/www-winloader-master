Инструкция

по установке Termux-box:
Шаг 1: Установка Termux
Скачайте и установите приложение Termux с f-droid.org.
Шаг 2: Установка Termux-x11
Скачайте и установите приложение Termux-x11 с GitHub.
Шаг 3: Установка Termux-box
Откройте Termux и выполните следующую команду:
curl -s -o x https://raw.githubusercontent.com/olegos2/termux-box/main/install && chmod +x x && ./x
Может потребоваться также ввести команду
termux-setup-storage
Дождитесь завершения установки (время может зависеть от скорости интернета).
Шаг 4: Запуск Termux-box
Для входа в главное меню Termux-box выполните команду:
termux-box
Вы можете выбрать режим эмулятора, выполнив соответствующую команду (1, 2, или 3).
Команда 5 откроет настройки.
Шаг 5: Запуск Termux-x11
После запуска Termux-x11, выберите "install" в меню "пуск/start".
Выберите DXVK и turnip для GPU Adreno 610 (или WineD3D и VirGL-Mesa для других процессоров).
GitHub репозиторий: github.com/olegos2/termux-box
